# Landing Page Images

Place your vintage camera images in this folder:

## Required Images:

1. **hero-bg.jpg** (1920x1080 recommended)
   - Hero section background image
   - Vintage camera collection or single vintage camera

2. **camera-1.jpg** (400x400 recommended)
   - First camera image in the "Discover Vintage Camera Excellence" section

3. **camera-2.jpg** (400x400 recommended)
   - Second camera image in the "Discover Vintage Camera Excellence" section

4. **about-camera.jpg** (600x600 recommended)
   - Image for the "About Digicam Shop" section
   - Vintage camera collection or shop display

5. **collection-camera.jpg** (600x600 recommended)
   - Image for the "Our Camera Collections" section
   - Premium vintage cameras

6. **reviewer.jpg** (400x400 recommended)
   - Reviewer/customer image for the reviews section
   - Photographer with camera (optional)

## Notes:
- Supported formats: JPG, PNG, GIF
- If images are not found, the page will automatically use external Unsplash URLs as fallback
- Optimize images for web (compress before uploading for better performance)
